# Uitlizando o DOM
- `getElementById` retorna o element
- `getElementByClassName` retorna o HTMLCollection
- `getElementByTagName` retorna o HTMLCollection
- `querySelector` retorna o element
- `querySelectorAll` retorna o NodeList
